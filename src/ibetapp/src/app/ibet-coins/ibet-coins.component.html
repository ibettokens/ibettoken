IBET Coins

Shared, Distributed, Dispersed


flash_on

Shared

Users of IBET tokens will receive bonus token for sharing it with others. It is designed to proliferate sharing. over all goal is to reach 22 million users.

group

Distributed

Contract owner only owns 1 coin at the beginning. only way to mint new coins is by sharing coins with others. this will continue until max supply token is shared.

settings

Dispersed

IBET coins are designed to be dispered among many users rather than just few people. the are rules in places as "sharing limit within 7 days" and others just to maintain a disperse set of users.



Steps to Earn and Share

looks_one
Request token from faucet

If you already have IBET tokens shared to you by someone, you should be all set and you can skip this step. If you do not have any IBET tokens to start sharing, you can click the button below to request starter token.


looks_two
Share token with people you like

Now that you have some tokens in your wallet, you can start sharing this token with your friends and family.

looks_3
Receive IBET Sharing bonus tokens

Once you have shared this token with new addresses, you will receive sharing BONUS IBET tokens.
